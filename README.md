# Superstore Sales & Profit Analysis Dashboard

## Overview

This Power BI dashboard provides insights into sales performance, profitability, customer segments, and product categories using the Superstore dataset.

## Tools Used

- Power BI
- Data Visualization
- Business Analytics
- Data Analysis

## Key Insights

- West region generated the highest sales.
- Technology is the most profitable category of products sold.
- Consumer customers account for nearly 48% of total sales.
- Phones, Chairs, and Binders are top-selling products.
